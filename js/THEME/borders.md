# Borders

## Width Scale

| Context | Width |
|---|---|
| Default (inputs, buttons, cards, badges, all components) | 4px |
| Emphasis / focus | 4px |

## Rules

- Use dotted borders by default — dotted is the primary border style across the entire system
- Solid borders only for dividers/separators inside components (e.g., accordion item separators, table row borders)
- Components in the same family must use matching border widths
- Never mix dotted and solid borders within a single component

## Usage

| Context | Width |
|---|---|
| Inputs / selects / textareas | 4px dotted; 4px dotted on focus or error (color changes) |
| Buttons | 4px dotted for all variants that have visible borders |
| Cards / containers | 4px dotted; consistent across all card types |
| Badges | 4px dotted |
| Alerts | 4px dotted |
| Modals | 4px dotted |
| Tables wrapper | 4px dotted |
| Accordion wrapper | 4px dotted |
| Tabs | 4px dotted |
| Dropdowns | 4px dotted |
